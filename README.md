## Super Loop example:
* file ADXL345.h
``` 
//Adxl345 Device Address 
#define adxl_address 0x53<<1

//Set i2c Handeler here
extern I2C_HandleTypeDef hi2c1;

```
* file main.c   
```
#include "ADXL345.h"
int16_t x, y, z;
int main()
{
  adxl_init()
  while (1)
  {
    x=adxl_readx();
		y=adxl_ready();
		z=adxl_readz();
  }  
}

```
